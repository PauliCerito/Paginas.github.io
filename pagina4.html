Planilla caso de uso 4: Iniciar sesión

Caso de uso Iniciar sesión
Identificador CU004
Actores Usuario registrado
Tipo Secundario
Referencias CU005 - CU006
Precondiciones Registrarse
Postcondiciones
Descripción El usuario inicia sesión
Resumen El usuario ingresa sus datos para poder ingresar al sistema
Flujo normal
Nro Actor Sistema
1 En la página principal, en la parte
superior derecha, hace clic en
“Iniciar sesión"
2 Se despliega una barra lateral pidiendo los siguientes datos:
- Nombre de usuario o mail
- Contraseña
3 Procede a completar los datos
4 Hace clic en “Acceder"
Flujo alternativo
2 La base de datos falla al verificar los datos.
2.1 Se le muestra un mensaje “Intente más tarde”.
