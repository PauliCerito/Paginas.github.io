Planilla caso de uso 6: Registrarse

Caso de uso Registrarse
Identificador CU006
Actores Usuario no registrado
Tipo Secundario
Referencias
Precondiciones Hacer consulta
Postcondiciones
Descripción El usuario se registra si así lo desea.
Resumen El usuario ingresa sus datos para que puedan ser almacenados en la base de dato y por ende el usuario entra al sistema
Flujo normal
Nro Actor Sistema
1 En la página principal, en la parte
superior derecha, hace clic en “Registrarse”.
2 Se abre una nueva pestaña mostrando las casillas para ingresar los datos:
- Nombre de usuario
- Email
- Repetir email
- Contraseña
- Repetir contraseña
- Número de teléfono
3 Completa los datos y hace clic en “Registrarse”
4 Se realiza la validación de los datos.
5 Los datos son guardados en la base de datos
6 Se le muestra página principal, con el chat, ya con su perfil creado
Flujo alternativo
4 Fallo al registrar los datos en la base de datos por falta de conexión a internet.
4.1 Se le muestra un mensaje “Por favor, intente más tarde”.
